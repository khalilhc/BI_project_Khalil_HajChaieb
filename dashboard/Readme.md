## **MarketFlow Attribution Dashboard**



**Overview:** Interactive Power BI dashboard for multi-channel marketing attribution analysis. 



##### **Dashboard Pages:**

###### **Executive Summary:**

**4 KPI cards (Spend, Revenue, ROAS, CAC):** 

* ROAS by channel performance
* Spend distribution
* Monthly trend analysis
* Date and channel filtering 



###### **Attribution Analysis:**

* Revenue comparison by channel
* Average sales cycle duration
* Comprehensive channel performance metrics
* Cross-page filtering enabled



###### **Geographic \& Forecast:**

* Geographic revenue distribution map
* 3-month revenue forecast with 95% confidence interval



###### **Channel Detail (Drill-Through):**

* Daily spend trends for selected channel
* Monthly metrics breakdown
* Accessible by right-clicking any channel
